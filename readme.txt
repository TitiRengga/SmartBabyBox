Deskripsi Smart Baby Box :
Pada jaman canggih sekarang ini, banyak orang tua yang kesusahan dalam menangani bayi mereka ketika sedang melakukan kegiatan rumah tangga, seperti menyapu, memasak, dan lain sebagainya. 
Oleh karena itu, dibutuhkan sebuah alat yang terhubung dengan aplikasi yang bernama Smart Baby Box, dimana pengguna harus menggunakan kode dari pembelian Smart Baby Box agar terhubung ke 
aplikasi dan kode yang digunakan berbeda setiap pembelian. Smart Baby Box memiliki fitur dimana apabila bayi berada di dalam Smart Baby Box, maka pengguna dapat menidurkan bayi dengan cara 
menentukan kekuatan dan waktu ayunan serta pengguna mendapatkan notifikasi apabila bayi di dalam Smart Baby Box menangis.

Pattern yang digunakan :
1. Singleton pattern untuk mengakses aplikasi smart baby box
2. Observer pattern untuk notifikasi apabila bayi menangis
3. Strategy pattern untuk kecepatan dari ayunan bayi.

untuk singleton pattern, dijalankan code menggunakan file babyBoxTest.java
untuk Observer pattern, dijalankan code menggunakan file smartBabyBoxTes.java
untuk Strategy pattern, dijalankan code menggunakan file speedTes.java
